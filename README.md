## What is readma.ru?
Simple reddit-like application.

### Prerequisites

* Ruby 2.6.5
* Yarn
* PostgreSQL 12
* ImageMagick

### Installation

* Make sure all the prerequisites are installed.
* Clone repository, ie. https://github.com/dev1vitaly/readmaru
* Run bin/setup
* Run rails s to start the application and head to http://localhost:3000/
